# Slack Integration

Connect your Alfe agent to Slack workspaces.

## Features

- **Channel messaging** — Respond to messages in Slack channels
- **Slash commands** — Register and handle slash commands
- **Thread support** — Manage threaded conversations

## Installation

```bash
alfe integration install slack
```

## Configuration

Connect your Slack workspace via OAuth — no manual token setup required.
