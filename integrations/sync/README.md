# Sync Integration

Back up your Alfe agent configuration and memory to the cloud.

## Features

- **Config backup** — Keep your agent configuration safe in the cloud
- **Conversation history** — Sync conversation logs for continuity
- **Memory sync** — Back up agent memory and learned context
- **Scheduled sync** — Configure automatic sync intervals

## Installation

```bash
alfe integration install sync
```

## Configuration

Choose what to sync and how often:
- **Sync scope**: Configuration, conversation history, and/or agent memory
- **Schedule**: Real-time, hourly, daily, or manual
