# Google Meet Integration

Join Google Meet calls as an AI participant.

## Features

- **Auto-join meetings** — Your agent automatically joins scheduled meetings
- **Voice participation** — Speak and listen in real-time during calls
- **Calendar integration** — Sync with Google Calendar for meeting awareness

## Installation

```bash
alfe integration install google
```

## Configuration

Connect your Google account via OAuth to grant Meet and Calendar access.
